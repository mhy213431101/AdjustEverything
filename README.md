# AdjustEverything

AdjustEverything 是一个面向测量平差学习与实验的 WinForms 原型项目。

当前版本主要实现了高程网平差的基础建模流程：

- 在画板上添加、拖动点
- 添加普通连线
- 添加高差观测
- 设置已知高程点
- 编辑点名、高程、高差观测值和中误差
- 删除点或高差观测
- 计算前检查高程网模型
- 执行高程网间接平差并输出结果

## 开发环境

- Visual Studio 2022
- .NET 8 SDK
- Windows

## 如何运行

1. 下载或克隆本仓库。
2. 用 Visual Studio 打开 `AdjustEverything.sln`。
3. 选择 `AdjustEverything` 项目。
4. 点击运行按钮，或按 `F5` 启动。
